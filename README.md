# Comment Category Prediction

## Overview
Built an NLP-based comment classification system for a Kaggle competition.

## Dataset
- Train: 198,000 samples
- Test: 102,000 samples

## Methods
- TF-IDF Vectorization
- Logistic Regression
- SVM
- Random Forest
- XGBoost
- LightGBM

## Results
- Best Macro F1 Score: 0.8121
- Best Model: Logistic Regression

## Tools
Python, Scikit-learn, Pandas, NumPy
